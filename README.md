# ChatCopyLite

ChatCopyLite is a lightweight addon for Ascension/WotLK that allows players to easily copy their chat log outside of the game.

## Features
- Small scroll icon near the chat
- Select all button
- CTRL+C to copy text
- No dependencies
- Clean and simple

## Installation
1. Download the latest release
2. Extract the folder into:  
   `World of Warcraft/Interface/AddOns/`
3. Enable **Load out of date addons** in the WoW client.
